# FutTabela - Dashboard de Futebol

Projeto desenvolvido para a disciplina de **Programação Web II** do IFCE - Campus Tauá.

## 📋 Visão Geral
O **FutTabela** é uma Single Page Application (SPA) que consome a API externa `football-data.org` para exibir tabelas de classificação, estatísticas e calendário de jogos das principais ligas do mundo. Além disso, conta com um utilitário de "Simulador de Racha" para sorteio de times amadores.

## 🚀 Tecnologias Utilizadas
* **React.js** (Vite)
* **JavaScript** (ES6+)
* **Axios** (Consumo de API REST)
* **CSS3** (Grid e Flexbox para responsividade)
* **Lucide React** (Ícones)

## 📂 Estrutura do Projeto
O projeto está organizado da seguinte forma:

* `/src/components` - Componentes reutilizáveis (Tabela, Jogos, Detalhes do Time).
* `/src/services` - Configuração da API (Axios).
* `/src/assets` - Imagens e recursos estáticos.

## ⚙️ Como Rodar o Projeto

1.  Clone este repositório:
    ```bash
    git clone https://github.com/guilhermealves05/fut-tabela.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
